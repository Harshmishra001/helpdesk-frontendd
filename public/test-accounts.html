Features by Role

  • Access to Dashboard with statistics
  • View and manage all tickets
  • View and manage all users

  • Update ticket status
  • Add notes to tickets

Agent Features:

  • View all tickets
  • Update ticket status
  • Add notes to tickets
  • View customer information

Customer Features:

  • View own tickets
  • Create new tickets
  • Add notes to own tickets
